How to Use

  1. Login to the dashboard using your credentials.
  2. On the Reports Overview page, click the "New Report" button.
  3. Upload your .json report file generated by DursGo.
  4. View the results across Overview, Scan Summary, and Vulnerabilities tabs.

Frequently Asked Questions

Why is my upload failing?

Ensure your file is a valid JSON file generated by DursGo. The file must contain scan_summary and vulnerabilities fields.

How do I delete a report?

Go to the Reports Overview (Home) page and click the "Delete" button next to the report you wish to remove.
